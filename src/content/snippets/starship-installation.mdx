# What is Starship?

Starship is cross-shell prompt that is compatitable with any shell and is very customizable in nature. It's also minimal and blazingly-fast.

# Prerequisities

1. **Terminal** (Like Windows Terminal for Windows)
2. [**Nerd Font**](https://www.nerdfonts.com/font-downloads) - Any font with nerd font patch.
