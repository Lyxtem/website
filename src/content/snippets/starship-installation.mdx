> Starship is cross-shell prompt that is compatitable with any shell and is very customizable in nature. It's also minimal and blazingly-fast.

# Prerequisities

1. **Terminal** of your choice
2. [**Nerd Font**](https://www.nerdfonts.com/font-downloads) - Any font with nerd font patch to get icons in terminal.

# Installation

Install it using any package manager you have on your machine.
